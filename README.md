# Gif Creator

Приложение для создания GIF-анимаций с поддержкой пиксельной эстетики, разработанное на Python с использованием PyQt6

## Особенности

- Загрузка изображений различных форматов.
- Настройка скорости и качества GIF.
- Предпросмотр анимации.
- Пиксельный интерфейс в светлых тонах.

## Требования

- Python 3.8+
- uv
- PyQt6
- Pillow

## Установка

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/hikara33/GIF-creator
    cd GIF-creator
    ```

2. Установите зависимости:
    ```bash
    uv sync
    ```

## Запуск

Запустите главное приложение:

```bash
python3 main.py
```

или

```bash
uv run python -m main
```
